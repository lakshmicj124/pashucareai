# Livestock Disease Detection

This is a full-stack web application designed to detect diseases in livestock (cows, goats, and sheep) using AI image classification.

## Folder Structure

- `frontend/`: React + Vite + Tailwind CSS application.
- `backend/`: Node.js + Express + PostgreSQL API server.
- `ai-model/`: Place for Tensorflow / Teachable Machine models and classification scripts.

## Getting Started

Detailed setup and execution instructions are available in the respective directories.
